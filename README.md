# Simple Articles Site

Готовый маленький сайт для публикации статей через веб-админку.

## Что внутри

- Главная страница со свежими статьями.
- Страница со всеми статьями.
- Отдельная страница каждой статьи.
- Админка по адресу `/admin`.
- Хранение статей в GitHub.
- Автоматическая публикация через Netlify.

## Самый простой запуск в интернете

1. Создайте новый репозиторий на GitHub.
2. Загрузите в него все файлы из этой папки.
3. Зайдите на Netlify и выберите `Add new site` -> `Import an existing project`.
4. Подключите GitHub и выберите созданный репозиторий.
5. Netlify сам увидит настройки из `netlify.toml`.
6. После публикации откройте в Netlify:
   `Site configuration` -> `Identity` -> `Enable Identity`.
7. Включите:
   `Services` -> `Git Gateway` -> `Enable Git Gateway`.
8. В `Identity` пригласите себя по email.
9. После принятия приглашения откройте:
   `https://ВАШ-САЙТ.netlify.app/admin`.

## Как публиковать статьи

1. Откройте `/admin`.
2. Нажмите `New Article`.
3. Заполните заголовок, краткое описание и текст.
4. Нажмите `Publish`.
5. Через короткое время статья появится на сайте.
